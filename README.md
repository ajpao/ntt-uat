# NTT-UAT — HA & MPIO UAT Simulator

Interactive single-page simulator สำหรับทดสอบและนำเสนอ User Acceptance Test (UAT) ด้าน High Availability (HA) และ Multi-Path I/O (MPIO)

## ภาพรวม
ไฟล์ uat.html เป็น standalone HTML ที่จำลองสถานการณ์ UAT สำหรับระบบ HA และ MPIO ในรูปแบบ dark-theme dashboard พร้อม animation ใช้สำหรับการ demo และ walkthrough กับลูกค้าหรือทีมงาน

## วิธีเปิดใช้งาน
- **Online:** [https://ajpao.github.io/ntt-uat/uat.html](https://ajpao.github.io/ntt-uat/uat.html)
- **Local:** เปิดไฟล์ uat.html ด้วย browser โดยตรง ไม่ต้องติดตั้งอะไรเพิ่ม

## เทคโนโลยีที่ใช้
- HTML5 / CSS3 / Vanilla JavaScript
- Google Fonts: Share Tech Mono, Noto Sans Thai
- ไม่มี external dependencies / framework

## หมายเหตุ
ไฟล์นี้เป็น UAT Simulator สำหรับใช้ภายในและการนำเสนอเท่านั้น
